# Smart Tourism Recommendation System

A comprehensive tourism platform that helps users plan trips, book accommodations, and explore destinations in Bangladesh.

## 🚀 Features

### User Features
- 🔐 **Authentication** - Register/Login with JWT
- 🎯 **Trip Planning** - Plan trips based on budget, duration, and interests
- 🏝️ **Destination Discovery** - Browse destinations with filters and search
- 🏨 **Hotel Selection** - View hotels with rooms, amenities, and pricing
- 🚌 **Transport Options** - Choose from various transport options
- 🍽️ **Food Categories** - Select meal plans based on budget
- 🎯 **Activities** - Add optional activities to trips
- 📅 **My Trips** - View and manage all trip plans
- 💰 **Booking System** - Book trips with payment simulation
- ⭐ **Ratings & Reviews** - Rate destinations and hotels

### Admin Features
- 📊 **Dashboard** - Overview of platform statistics
- 👥 **User Management** - Manage user accounts and roles
- 🏝️ **Destination Management** - CRUD operations for destinations
- 🏨 **Hotel Management** - Manage hotels and rooms
- 🚌 **Transport Management** - Manage transport options
- 🍽️ **Food Categories** - Configure meal plans
- 🎯 **Activities Management** - Add/Edit activities
- 📅 **Booking Overview** - View and manage bookings
- 💰 **Revenue Reports** - Financial analytics and exports
- ⭐ **Ratings Moderation** - Manage user reviews

## 🛠️ Tech Stack

### Frontend
- React.js
- Tailwind CSS
- React Router DOM
- Chart.js (for analytics)

### Backend
- Node.js
- Oracle Database
- JWT Authentication
- Multer for file uploads

## 📋 Prerequisites

- Node.js (v14 or higher)
- Oracle Database
- npm or yarn
